## 已知限制

- 当前任务栏是启动时捕获的静态视觉，不可交互，也不会刷新时钟和托盘状态。
- 当前重点支持主显示器和 Windows 默认 Shell；复杂多显示器、第三方 Shell 和特殊 DPI 组合仍需验证。
- 第一版只有全部图标觉醒和自由漫游，不包含打架动作。
- 轻微演员重叠不作为 RC1 阻塞问题。
- 当前没有安装器、自动更新和代码签名。
