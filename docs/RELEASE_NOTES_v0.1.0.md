# Besktop v0.1.0 发布说明（候选稿）

Besktop 是一个安全的 Windows 桌面娱乐工具。运行后，你熟悉的桌面图标会突然醒来，长出白色手脚，在桌面上自由闲逛；退出后，真实桌面保持原样。

## 这一版有什么

- 重建当前桌面的壁纸、图标、文字和任务栏静态画面。
- 让捕获到的全部桌面图标从真实原位错峰觉醒。
- 为每个图标长出白色四肢，并在工作区域内自由漫游、停留和转向。
- 支持普通文件、文件夹、快捷方式、缩略图，以及“此电脑”“回收站”等虚拟 Shell 项的真实图标。
- 漫游区域避开系统报告的任务栏工作区。

当前版本先做好“全部图标觉醒和自由漫游”，暂不包含拳击、踢击或其他打架动作。

## 安全边界

所有演出都发生在 Besktop 自己的全屏舞台中。Besktop 不会移动、隐藏、删除或重命名真实桌面文件，不会控制 Explorer，也不会修改真实任务栏。任务栏画面只在启动时静态捕获，不可点击，也不会持续刷新时钟或托盘状态。

随时可以按 `Esc` 或 `Ctrl+Shift+B` 退出。

## 使用方式

1. 下载单个 `Besktop.exe`。
2. 双击运行，无需安装。
3. 观看桌面图标醒来并闲逛。
4. 按 `Esc` 或 `Ctrl+Shift+B` 退出。

当前版本没有安装器和自动更新，也尚未进行代码签名。首次运行时，Windows 可能显示安全提示；请确认文件来自项目官方发布页，并核对正式 Release 页面提供的 SHA-256。不要关闭安全软件或绕过其检测。

## 支持范围

当前重点验证 Windows 11 与 Windows 10 LTSC 1809 的 x64 环境、主显示器和 Windows 默认 Shell。不同 DPI、任务栏自动隐藏、特殊任务栏位置、复杂多显示器和第三方 Shell 的表现仍在持续验证，不承诺所有组合已经兼容。

## 已知限制

- 任务栏是启动时的静态画面，时间、托盘和按钮状态不会刷新，也不能交互。
- 第一版不包含打架动作。
- 演员之间可能偶尔轻微重叠。
- 当前没有安装器、自动更新和代码签名。
- 全屏程序、远程桌面、录屏软件和特殊显示配置可能需要额外兼容性验证。
